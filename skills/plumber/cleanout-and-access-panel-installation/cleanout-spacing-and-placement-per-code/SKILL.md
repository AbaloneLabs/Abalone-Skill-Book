# Cleanout Spacing and Placement Per Code

Cleanouts are the access points that make a drainage system serviceable, and their spacing and placement are governed by code because a system without enough cleanouts, or with cleanouts in the wrong locations, forces a future blockage to be cleared by breaking walls, cutting pipe, or excavating the building sewer, all of which are catastrophically more expensive than installing a cleanout during rough-in. The judgment problem is that the IPC and UPC specify cleanout locations with precise rules (at the base of every sanitary stack, at every change of direction greater than 45 degrees, at maximum intervals along the building drain and sewer, at the building sewer exit), and a plumber who installs cleanouts by habit or convenience rather than by tracing the code requirements will leave gaps that only surface years later when a blockage cannot be reached. Sizing matters too: cleanouts must meet minimum diameters (not smaller than one pipe size below for small pipe, and 4 inch for the building drain in many jurisdictions), and access clearance must allow a rodding machine to couple to the cleanout. This skill covers the spacing, placement, sizing, and clearance decisions that make a drainage system maintainable.

## Core Rules

### Place Cleanouts at Every Code-Mandated Location, Tracing the Full System

The code-required cleanout locations are non-negotiable and must be traced through the entire drainage system, not installed selectively. The IPC and UPC require a cleanout at the base of every sanitary stack (the transition from vertical to horizontal), at every change of direction in the building drain greater than 45 degrees (cumulative, so two 45-degree fittings that total 90 degrees require a cleanout), at the upstream end of the building drain (the highest point of the horizontal main), at the connection of the building drain to the building sewer, and at intervals along the building sewer. The trap is installing cleanouts at the obvious points (base of the main stack) and omitting them at the less obvious ones (an intermediate change of direction under a slab, a stack serving an upper floor), which leaves a future blockage unreachable. The disciplined rule is to walk every horizontal run and every stack transition, identify each code-required cleanout location, and install a cleanout at each.

### Space Cleanouts Along the Building Drain and Sewer at Code Intervals

In addition to the location-specific cleanouts, the building drain and building sewer require cleanouts at maximum intervals so that no segment of buried or concealed piping is beyond rodding reach. The IPC specifies cleanouts every 100 feet for piping 4 inches and larger and every 50 feet for piping smaller than 4 inches (verify the adopted code and edition; the UPC has its own intervals and the local amendment may tighten them). The interval is measured along the pipe run, and where a run exceeds the interval without a natural cleanout location, an intermediate cleanout must be added. The trap is treating the 100-foot (or 50-foot) interval as a maximum only on paper while omitting intermediate cleanouts on a long run because "it is straight and unlikely to clog," which leaves the middle of the run unreachable. The disciplined rule is to measure each run, add intermediate cleanouts where the interval is exceeded, and document the spacing.

### Size Cleanouts to the Minimum Code Diameter for the Pipe Served

Cleanout size affects rodding reach: a small cleanout admits only a small cable, limiting how far a blockage can be cleared. The IPC and UPC require cleanouts to be the same size as the pipe they serve up to 4 inches, and for pipe larger than 4 inches the cleanout must be not less than 4 inches (verify the adopted code; some editions and amendments differ for very large pipe). For small pipe, the cleanout must not be smaller than one pipe size below the pipe served, and in no case smaller than the minimum code cleanout size. The trap is installing a 2-inch cleanout on a 3-inch or 4-inch line to save a fitting, which limits rodding reach and may fail inspection. The disciplined rule is to size every cleanout to the code minimum for the pipe served, defaulting to matching the pipe size up to 4 inches, and to document the sizing.

### Provide Access Clearance for Rodding Equipment and Removal

A cleanout behind a finished wall or under a covered floor is useless unless an access panel or box of adequate size is provided, and the code requires minimum clearance in front of each cleanout for the rodding equipment to couple and operate. The IPC and UPC generally require at least 18 inches of clearance in front of the cleanout (measured from the cleanout face) for the largest size of cleanout, and the access opening must be large enough for the plug to be removed and the rodding machine to be coupled. Cleanouts buried under tile, concealed behind cabinetry, or boxed in by furred-out walls without an access panel are effectively absent. The trap is installing the cleanout correctly but leaving no access, so the first service call requires demolition. The disciplined rule is to provide a code-sized access panel or box at every concealed cleanout and to verify the 18-inch clearance in front.

### Use Two-Way Cleanouts at the Base of Stacks and at the Building Sewer Exit

A two-way (double) cleanout allows rodding both downstream (with the flow, toward the sewer) and upstream (against the flow, back toward the stack), which is essential at the base of a stack and at the building sewer exit because blockages can occur in either direction. A two-way cleanout is a Y-fitting with cleanout plugs on both the upstream and downstream branches, installed so that rodding can be directed either way. The trap is installing a single-direction cleanout at the base of a stack that allows rodding only downstream, leaving an upstream blockage in the stack unreachable without cutting pipe. The disciplined rule is to use a two-way cleanout at the base of every sanitary stack and at the building sewer exit, and to orient it so both directions are accessible.

## Common Traps

### Omitting Cleanouts at Intermediate Direction Changes

A plumber installs cleanouts at the base of the main stack and at the sewer exit but omits them at an intermediate 90-degree change of direction under the slab. The trap is that the change of direction is exactly where blockages accumulate, and without a cleanout it is unreachable. The mechanism is that fittings over 45 degrees (cumulative) require a cleanout. The false signal is that "the main cleanouts are installed." The harm is excavation or pipe cutting to reach the blockage. The defense is to trace every horizontal run and install a cleanout at each change of direction over 45 degrees.

### Treating the 100-Foot Interval as a Maximum Only on Paper

A plumber runs a 140-foot straight building sewer with cleanouts only at the ends, omitting the intermediate cleanout the interval requires. The trap is that the middle of a long straight run is the hardest point to rod. The mechanism is that cable reach and friction limit rodding distance. The false signal is that "the run is straight, so it will not clog." The harm is an unreachable blockage requiring excavation. The defense is to measure each run and add intermediate cleanouts where the code interval (100 ft for 4 inch and larger, 50 ft for smaller) is exceeded.

### Undersizing the Cleanout to Save a Fitting

A plumber installs a 2-inch cleanout on a 3-inch or 4-inch line to avoid buying a larger fitting. The trap is that the small cleanout limits rodding reach and may violate the code sizing rule. The mechanism is that cleanout size governs the cable size that can be inserted. The false signal is that "a cleanout is installed." The harm is a blockage that cannot be cleared and an inspection failure. The defense is to size every cleanout to the code minimum (matching the pipe size up to 4 inches) and to document the sizing.

### Installing a Cleanout Without Access Clearance or an Access Panel

A plumber installs a cleanout correctly in the wall but leaves no access panel, so the cleanout is concealed behind drywall. The trap is that a concealed cleanout is functionally absent. The mechanism is that the code requires both the cleanout and access to it. The false signal is that "the cleanout is there per the drawings." The harm is drywall demolition at the first service call. The defense is to provide a code-sized access panel and at least 18 inches of clearance in front of every concealed cleanout.

### Using a Single-Direction Cleanout Where a Two-Way Is Required

A plumber installs a single-direction cleanout at the base of a stack, allowing rodding only downstream, and an upstream blockage in the stack is unreachable. The trap is that the base of a stack and the sewer exit need rodding in both directions. The mechanism is that blockages can occur upstream or downstream of the fitting. The false signal is that "a cleanout is at the base of the stack." The harm is pipe cutting to reach an upstream blockage. The defense is to use a two-way cleanout at the base of every stack and at the building sewer exit, oriented for access in both directions.

## Self-Check

- Did I trace every horizontal run and every stack transition and install a cleanout at each code-required location (base of every stack, change of direction over 45 degrees cumulative, upstream end of the building drain, building drain-to-sewer connection)?
- Did I measure each run and add intermediate cleanouts where the code interval is exceeded (100 ft for 4 inch and larger, 50 ft for smaller per the IPC; verify the adopted code)?
- Did I size every cleanout to the code minimum (matching the pipe size up to 4 inches; not smaller than one pipe size below for small pipe)?
- Did I provide a code-sized access panel or box at every concealed cleanout, with at least 18 inches of clearance in front?
- Did I use a two-way cleanout at the base of every sanitary stack and at the building sewer exit, oriented for rodding in both directions?
- Did I verify that no cleanout is concealed behind cabinetry, under fixed finishes, or boxed in without access?
- Did I confirm the cleanout plug is the correct type (countersunk or raised) for the location (traffic vs non-traffic)?
- Did I coordinate cleanout locations with the architect and owner so access panels are aesthetically and functionally placed?
- Did I document each cleanout location, size, type, and access provision so the system is verifiably serviceable?
- Did I confirm the cleanout spacing and sizing against the adopted code edition and any local amendments before calling for inspection?
