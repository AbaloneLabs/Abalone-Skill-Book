# Cross-Format Adaptation

Cross-format adaptation is the craft of reshaping content from one format (long video, essay, podcast, newsletter) into another (carousel, thread, short-form clip, infographic) so that it serves the new format's audience and norms genuinely. Adaptation is not conversion: the same value expressed in a new format requires rethinking structure, pacing, depth, and presentation, because what works in one format often fails in another. The harm this skill prevents is the common pattern of mechanical format conversion, where a video script is pasted into a thread or a essay is sliced into carousel slides verbatim, producing content that feels alien to the format and underperforms because it ignores how each format's audience actually consumes.

Use this skill when adapting any piece across formats, when repurposed content underperforms in its new format, and when building templates for recurring cross-format work.

## Core Rules

### Understand Each Format's Consumption Logic

Each format has a distinct consumption logic that determines what works. Before adapting, understand the target format:

- Long video rewards structure, narrative, and sustained value across minutes.
- Short-form video rewards immediate hooks, fast pacing, and a single clear payoff in under a minute.
- Carousels reward visual progression, one idea per slide, and a satisfying arc across swipes.
- Threads reward punchy openings, one idea per post, and cumulative build.
- Newsletters reward depth, voice, and a complete reading experience.
- Podcasts reward conversation, audio clarity, and sustained attention without visual aids.
- Infographics reward visual hierarchy, data clarity, and scannability.

Adapting without understanding the target format's logic produces content that fights the format. Study what performs in the target format before adapting into it.

### Identify The Core Value To Carry Across

Cross-format adaptation begins by identifying the core value of the source: the single insight, framework, story, or answer that makes the source worth adapting. Everything else, examples, asides, supporting detail, is negotiable. Carry the core value across; reshape everything around it to serve the target format.

If the source has multiple distinct core values, consider adapting each into a separate piece rather than cramming all into one adapted piece. One core value per adapted piece serves the format better than compressing several.

### Restructure For The Target Format, Do Not Preserve The Source Structure

The source's structure serves the source's format. The adapted piece needs a structure that serves the target format. A long video's minute-by-minute structure does not map to a carousel's slide-by-slide structure; an essay's paragraph flow does not map to a thread's post-by-post flow. Restructure from the core value outward:

- decide the adapted piece's hook, body, and payoff based on the target format;
- choose which source elements support the core value in the new structure;
- cut source elements that do not serve the adapted piece, even if they were valuable in the source;
- add new elements the target format requires that the source lacked.

Preserving the source structure produces format-mismatched content. Restructuring produces format-native content.

### Calibrate Depth And Length To The Format

Each format implies a depth and length range that the audience expects. A carousel cannot carry the depth of a 30-minute video; a newsletter should not be as thin as a short-form clip. Calibrate the adapted piece's depth and length to what the format supports and the audience expects:

- short formats carry one core idea, sharply;
- medium formats carry a core idea with supporting structure;
- long formats carry depth, nuance, and multiple supporting elements.

Attempting to force source depth into a format that cannot hold it produces either cramped, overwhelming pieces or shallow extractions that lose the value. Match depth to format.

### Rewrite For The Format's Voice And Pacing

Even when the source and adapted piece share the creator's voice, the pacing and register differ by format. A long video script's spoken sentences may be too long for a thread's punchy posts; an essay's formal paragraphs may be too dense for a carousel's slide text. Rewrite for the format:

- adjust sentence length and rhythm to the format's pacing;
- adjust register (spoken vs. written, formal vs. casual) to the format's norms;
- adjust the density of information per unit (per slide, per post, per second) to what the format's audience can absorb.

Verbatim reuse of source text almost never produces format-native pacing. Rewrite.

### Add Format-Appropriate Visual And Structural Elements

Formats have signature elements the adapted piece should use: carousels use slide titles, visual progression, and a closing slide; threads use numbering, hooks, and a summary post; short-form uses text overlays, captions, and cuts; infographics use hierarchy, icons, and data visualization. Add the elements the target format expects, sourced from the core value.

Adapted pieces that omit format signatures feel alien to the format's audience. Use the format's native elements.

### Preserve Accuracy And Context In Adaptation

Adaptation can distort the source's meaning by removing context. A nuanced source claim that included conditions or caveats can become an absolute claim when extracted without its qualifications. Preserve accuracy:

- carry conditions and caveats relevant to the extracted value;
- avoid extracting claims that depended on context the new format omits;
- if a claim cannot be made accurately in the shorter format, choose a different value to adapt or add the necessary context;
- cite sources and data where the adapted piece makes factual claims.

Accuracy in adaptation is as important as accuracy in the original, because audiences encounter adapted pieces as standalone claims.

### Test Adapted Pieces Against Format Performance

Adapted pieces should be evaluated against the target format's performance norms, not against the source's performance. A well-adapted carousel is judged by carousel metrics (saves, swipes, shares), not by whether it matched the source video's views. Track adapted piece performance by format-appropriate metrics and learn which adaptations work in which formats for this audience.

## Common Traps

### Mechanical Conversion

Pasting source content into a new format without restructuring produces format-mismatched content that underperforms. Restructure from the core value outward.

### Preserving Source Structure

Mapping the source's structure onto the target format ignores the target format's logic. Restructure for the target format.

### Depth Mismatch

Forcing source depth into a format that cannot hold it, or extracting too shallowly for a format that expects depth, produces mismatched pieces. Calibrate depth to format.

### Verbatim Text Reuse

Reusing source text without rewriting for the format's pacing and register produces alien-feeling content. Rewrite for the format.

### Omitting Format Signatures

Adapted pieces that lack the target format's native elements (slide titles, thread numbering, overlays) feel foreign to the format's audience. Use format-native elements.

### Context Stripping

Extracting claims without their conditions or caveats turns nuanced sources into misleading adapted pieces. Preserve accuracy and context.

### Cramming Multiple Values Into One Piece

Compressing several source values into one adapted piece overwhelms the format and dilutes each value. Adapt one core value per piece.

### Judging Adapted Pieces By Source Metrics

Evaluating adapted pieces against the source's performance rather than format-appropriate metrics misleads optimization. Use format-appropriate metrics.

### Irresponsible Adaptation Of High-Stakes Content

Adapting health, financial, legal, mental health, or safety content into shorter formats can strip the caveats, disclosures, and hedging that made the source responsible, producing confident-sounding claims that constitute misleading or harmful advice. Preserve all relevant disclosures and limits in adaptation, add context where the format requires it, and where adapted content approaches regulated advice, escalate to expert review and professional referral. If a claim cannot be made responsibly in the shorter format, do not adapt it.

## Self-Check

Before approving a cross-format adaptation, verify:

- The target format's consumption logic is understood before adapting.
- The core value to carry across is identified, with one core value per adapted piece.
- The adapted piece is restructured for the target format, not preserving the source structure.
- Depth and length are calibrated to what the format supports and the audience expects.
- Text is rewritten for the format's pacing, register, and information density, not reused verbatim.
- Format-native elements (slide titles, numbering, overlays, hierarchy) are added.
- Accuracy and context are preserved; conditions, caveats, and sources relevant to the extracted value are carried over.
- Adapted pieces are evaluated against format-appropriate performance metrics, not source metrics.
- For high-stakes content, disclosures and limits are preserved across adaptation, context is added where needed, and expert review or professional referral is used where adapted content approaches regulated advice.
