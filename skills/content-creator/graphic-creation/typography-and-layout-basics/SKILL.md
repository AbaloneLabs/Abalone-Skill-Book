# Typography And Layout Basics

Typography and layout are the quiet infrastructure of every visual asset. When they are right, the audience does not notice them; they simply read and understand. When they are wrong, the audience feels confusion, clutter, or cheapness without being able to name the cause. The judgment problem is that type and layout decisions look subjective but are mostly governed by legibility, hierarchy, rhythm, and restraint. Creators routinely fail by choosing fonts for personality instead of function, by using inconsistent spacing that makes layouts feel amateur, or by over-decorating type until it stops communicating.

The harm is real. Poor hierarchy buries the message. Inconsistent spacing signals carelessness. Decorative type choices date quickly and undermine credibility. This skill helps the agent make type and layout decisions that serve the message, hold up across assets, and feel deliberate.

## Core Rules

### Establish Hierarchy Before Choosing Fonts

Hierarchy is the contract between the design and the reader about what to read first, second, and third. Decide the hierarchy before touching fonts, because font choices should express an already-decided structure, not invent it.

Define at minimum:

- the primary level, usually the headline or main claim;
- the secondary level, subheads, pull quotes, or section labels;
- the body level, the readable running text;
- the utility level, captions, dates, footnotes, disclaimers.

Each level should differ clearly in size, weight, or treatment so the reader's eye moves in the intended order. If two levels look similar, the hierarchy is not doing its job.

### Pair Fonts For Function And Contrast, Not Novelty

Most layouts need at most two typefaces, sometimes one. The goal of pairing is contrast of role, not collection of personalities. A common strong pattern is a distinctive display face for headlines paired with a neutral, highly legible text face for body.

When pairing, consider:

- contrast in classification, for example a serif headline with a sans-serif body, or vice versa;
- contrast in weight and proportions so the roles do not blur;
- shared mood or era so the pair feels coherent rather than fighting;
- legibility of the body face at small sizes and long lengths.

Avoid pairing two loud display faces, which compete and produce noise. One voice should lead; the other should support.

### Respect Legibility As A Hard Constraint

No amount of style justifies text the audience cannot read. Legibility is a functional requirement, especially for body text and small UI elements.

Guard legibility by:

- keeping body text at sizes that read comfortably on the target surface;
- ensuring sufficient contrast between text and background;
- choosing body faces with open counters and distinct letterforms;
- avoiding all-caps for long runs of text;
- setting line length and leading so the eye can track lines without strain.

Decorative faces belong in headlines and short labels, never in paragraphs.

### Build A Grid And Defend Consistent Spacing

Consistent spacing is what separates a deliberate layout from an accidental one. A grid gives every element a predictable relationship to every other element, and reusable spacing values prevent the drift that makes feeds look messy.

Establish and reuse:

- a base spacing unit and a small set of multiples;
- consistent margins and gutters;
- alignment rules, left, center, or grid-based;
- vertical rhythm between levels of hierarchy.

When every margin is a different value, the layout feels handmade in the worst way. Define spacing once and apply it everywhere.

### Use White Space As An Active Element

White space is not empty space waiting to be filled; it is a tool that directs attention, creates emphasis, and signals confidence. Crowding every region of a layout dilutes the message and signals insecurity.

Use white space to:

- isolate the primary element so it dominates;
- separate sections so structure is readable;
- give the design room to breathe at small display sizes;
- imply quality and intentionality.

If a layout feels busy, the first move is usually to remove and to add space, not to add more elements.

### Limit Type Sizes And Weights To A Small Set

A common amateur signal is using a different size or weight for every element. Professional layouts reuse a small, defined set of type styles.

Define a type scale and stick to it:

- a fixed set of sizes mapped to hierarchy levels;
- a fixed set of weights, usually two or three;
- consistent treatment of each role across all assets.

This discipline is what makes a multi-post feed or a multi-page deck feel like one designed system rather than a pile of unrelated pages.

### Align Deliberately And Avoid Accidental Alignment

Alignment is invisible when correct and jarring when wrong. Mixed or partial alignment creates visual tension the audience reads as sloppiness.

Choose alignment intentionally:

- pick a dominant alignment, usually left for body text in Latin scripts;
- align related elements to a common axis;
- avoid centering long blocks of body text, which is hard to read;
- make sure logos, captions, and labels follow the same grid as the rest.

When two elements are almost but not quite aligned, the eye notices. Align fully or offset clearly.

### Match Type Mood To Content And Audience

Type carries tone before the words are read. A playful rounded face undermines serious content; a stiff corporate face undermines warm personal content. Match the voice of the type to the voice of the message.

Consider:

- the audience's expectations and reading context;
- the emotional register of the content;
- whether the face will date quickly due to a trend;
- whether the face supports the languages and character sets you need.

Trendy faces feel fresh now and dated soon. Neutral, well-drawn faces tend to age better.

## Common Traps

### Choosing Fonts For Personality Over Function

A face that expresses mood but fails at legibility or hierarchy hurts the layout. Function leads; personality supports.

### Using Too Many Typefaces

Three or four faces in one layout compete and produce noise. One or two well-paired faces almost always suffice.

### Inconsistent Spacing And Margins

When every gap is a different value, the layout feels accidental. A defined spacing system prevents this drift.

### Filling All Available Space

Treating white space as waste produces crowded, low-confidence layouts. Space is a design element, not a vacancy.

### A Different Size Or Weight For Every Element

Ad hoc type sizing signals amateur work. Reuse a small defined type scale across all assets.

### Decorative Faces In Body Text

Display faces used for paragraphs destroy legibility. Reserve them for headlines and short labels.

### Trend-Driven Pairings That Date Fast

Choosing a face because it is currently popular ages the design quickly. Prefer durable, well-constructed type.

## Self-Check

Before approving a typographic layout, verify:

- Hierarchy levels are defined and clearly distinguishable in size, weight, or treatment.
- No more than two typefaces are used, and they contrast in role rather than compete.
- Body text is legible at the size and contrast it will actually be read.
- A consistent spacing system is applied, with no ad hoc margin values.
- White space is used intentionally to isolate the primary element and separate sections.
- A small, fixed type scale is reused across the asset or feed.
- Alignment is deliberate and consistent on a shared axis.
- The type mood matches the content and audience without relying on a fast-aging trend.
- Display or decorative faces appear only in headlines or short labels, never in body text.
- The layout reads in the intended order when viewed cold by someone who has not seen it before.
