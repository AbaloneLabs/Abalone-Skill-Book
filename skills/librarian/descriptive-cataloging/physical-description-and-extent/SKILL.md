# Physical Description And Extent

The physical description area of a catalog record seems simple: record what the thing is and how big it is. In practice it is one of the most detail-laden and consequential parts of description. The physical description tells a patron whether an item will fit on their shelf, whether it has the illustrations they need, whether the audio recording is the right format, and whether the digital file will run on their system. It supports identification, distinguishing editions and impressions; it supports access, telling patrons what equipment or conditions they need; and it supports management, feeding shelving, preservation, and circulation decisions. The trap is to treat the physical description as a perfunctory transcription of the colophon, when it actually requires judgment about which details matter, how to record multipart and incomplete items, how to handle accompanying material, and how to balance precision against practicality. A sloppy physical description misleads patrons about what they will actually receive and complicates every downstream use of the record.

Use this skill when describing physical description and extent, when handling multipart items and incomplete holdings, when recording format-specific characteristics, and when deciding how to treat accompanying material. The goal is to prevent the agent from treating the physical description as perfunctory, from omitting access-critical details, from mishandling incomplete or multipart items, or from recording extent in ways that confuse rather than inform.

## Core Rules

### Understand The Purpose Of The Physical Description

The physical description serves multiple purposes beyond mere transcription. Keep them all in view.

It supports:

- **Identification:** distinguishing editions, impressions, and formats.
- **Access:** telling patrons what equipment, conditions, or formats they need.
- **Management:** feeding shelving, preservation, and circulation decisions.
- **Use:** indicating illustrations, sound, color, duration, and other use-relevant features.

Recording physical details with all these purposes in mind produces records that actually serve users.

### Record Extent Accurately And Meaningfully

Extent records the quantity of the resource. Record it in the prescribed unit and with meaningful precision.

Record:

- the number of pages, volumes, discs, reels, files, or items;
- the correct unit for the format (pages, leaves, volumes, pieces, files);
- the prescribed form for last-page numbering (e.g., recording the last numbered page);
- unnumbered sequences where significant;
- for multipart items, the total extent where known or the extent of the part described.

Accurate extent supports identification and management; imprecise extent misleads.

### Handle Multipart And Incomplete Items Carefully

Multipart monographs and incomplete holdings are common and require careful extent recording.

Handle by:

- recording the extent of the part in hand for analytical description;
- recording the total extent where the complete set is described;
- noting incomplete holdings clearly in holdings statements;
- distinguishing between a complete multipart item and a single volume of one;
- using appropriate notes for analyzed or partially held sets.

Confusing part and whole extent misleads patrons about what is available.

### Record Dimensions Where Significant

Dimensions matter for shelving, storage, and use. Record them where prescribed and significant.

Record:

- height of books in centimeters, following the prescribed rounding;
- dimensions for non-standard formats (folios, miniatures, oblong);
- duration for time-based media;
- file size or resolution where prescribed for digital resources;
- dimensions for maps, objects, and other format-specific resources.

Omitting significant dimensions complicates shelving and use.

### Record Format-Specific Details

Each format has physical and technical details that support access and use. Record them.

Examples:

- **Books:** illustrations, color, accompanying material, format (large print, braille).
- **Sound recordings:** duration, sound type (mono, stereo), recording format.
- **Video:** duration, sound, color, aspect ratio, video format.
- **Maps:** scale, projection, color, folding.
- **Digital files:** file type, encoding, system requirements.
- **Objects:** material, technique, markings.

Format-specific details are often the difference between a usable and unusable record.

### Describe Accompanying Material Appropriately

Accompanying material, booklets, inserts, maps, supplements, and digital extras, must be described. Choose the right approach.

Approaches:

- note accompanying material in the physical description or a note;
- describe substantial accompanying material as part of the extent;
- create analytical records for significant separate components;
- note the relationship so patrons understand what is included.

Silently ignoring accompanying material hides content patrons may need.

### Apply Consistent Units And Conventions

Consistency in units and conventions makes records predictable and comparable. Follow standards.

Apply:

- prescribed units for each format (pages, centimeters, minutes, files);
- prescribed rounding for dimensions;
- standard abbreviations and punctuation;
- consistent treatment of similar items across the catalog.

Inconsistency makes the catalog harder to use and to manage.

### Balance Precision Against Practicality

Physical description can be pursued to infinite precision. Balance detail against practical value.

Balance by:

- recording details that support identification, access, and management;
- omitting trivial details that do not serve any purpose;
- using notes for exceptional or unusual characteristics;
- following local and national standards for the level of detail expected.

Over-precise records burden catalogers without aiding users; under-precise records mislead.

### Note Physical Condition Where It Affects Access

Physical condition can affect access and use. Note it where relevant.

Note:

- significant damage or fragility that limits handling;
- restrictions due to preservation status;
- missing components or plates;
- facsimile or reproduction status where it affects use.

Condition notes protect fragile material and set patron expectations accurately.

### Coordinate With Holdings Statements

The physical description and holdings statements must agree. Coordinate them.

Coordinate:

- extent described with the actual holdings recorded;
- multipart descriptions with the volumes actually held;
- accompanying material noted with what is actually present;
- gaps and missing parts reflected in holdings.

Disagreement between description and holdings misleads patrons about availability.

## Common Traps

### Treating Physical Description As Perfunctory

Physical details support identification, access, and management. Record them with purpose.

### Recording Extent Imprecisely Or In The Wrong Unit

Use the correct unit and meaningful precision for the format.

### Confusing Part And Whole For Multipart Items

Distinguish the part in hand from the complete set, and note incomplete holdings.

### Omitting Significant Dimensions

Dimensions matter for shelving and use. Record them where prescribed.

### Skipping Format-Specific Details

Duration, sound, color, scale, and file type are access-critical. Record them.

### Ignoring Accompanying Material

Booklets, inserts, and supplements are often important. Describe them appropriately.

### Inconsistent Units And Conventions

Follow prescribed units, rounding, and abbreviations for predictability.

### Description Disagreeing With Holdings

Extent and holdings must agree. Coordinate them to avoid misleading patrons.

## Self-Check

- Did you record physical details with identification, access, management, and use in mind?
- Did you record extent accurately in the correct unit with meaningful precision?
- Did you handle multipart and incomplete items carefully, distinguishing part from whole?
- Did you record dimensions where significant, using prescribed units and rounding?
- Did you record format-specific details (illustrations, sound, color, duration, scale, file type)?
- Did you describe accompanying material appropriately, in notes, extent, or analytical records?
- Did you apply consistent units, conventions, and abbreviations across similar records?
- Did you balance precision against practicality, recording details that serve a purpose?
- Did you note physical condition where it affects access or handling?
- Does the physical description agree with the holdings statements for the item?
