# Bibliographic Completeness

The bibliography or reference list is the work's scholarly apparatus, the record of sources on which the work depends. A complete bibliography contains every source used, with every required element for each entry, allowing readers to verify and explore the work's foundation. An incomplete bibliography, missing sources, missing elements, or inconsistent documentation, undermines the work's traceability and signals carelessness. Bibliographic completeness is the discipline of verifying that the reference apparatus is comprehensive and fully documented. Editors who check citation format but not bibliographic completeness produce works whose scholarly apparatus has gaps that prevent verification and erode credibility.

Use this skill when editing works with a bibliography or reference list, including academic papers, books, theses, and research-based nonfiction. It covers verifying all sources are listed, all required elements are present, and the apparatus is internally complete. The goal is a reference list that fully and consistently documents the work's sources.

## Core Rules

### Verify Every Cited Source Appears In The Bibliography

The most basic completeness requirement is that every source cited in the text appears in the bibliography or reference list. Orphan citations, cited but not listed, are a clear completeness failure that prevents verification. Verify every cited source is listed.

Cross-reference every in-text citation against the bibliography. Flag any citation without a corresponding entry. Common causes include sources added late in revision without updating the bibliography, citations that were changed without updating the list, and simple oversight. Every orphan citation must be resolved by adding the entry or removing the citation. Complete cross-referencing ensures the bibliography contains at minimum every source the work relies on through citation.

### Verify Every Listed Source Is Cited In The Text

The reverse check is that every source listed in the bibliography is actually cited in the text. Orphan references, listed but not cited, inflate the bibliography and may indicate sources that were consulted but not used, or citations that were removed without updating the list. Verify every listed source is cited.

Cross-reference every bibliography entry against the in-text citations. Flag entries that are not cited anywhere in the text. Determine whether the entry should be cited, was cited and the citation was lost, or should be removed. Some works include a bibliography of consulted works beyond those cited, but this should be a deliberate choice, not an oversight. Resolve orphan references by citing, removing, or explicitly designating them as consulted-only. Symmetric completeness ensures the citation apparatus and bibliography agree.

### Verify All Required Bibliographic Elements Are Present

Each reference entry must contain all the elements the style guide requires for its source type: author, date, title, source, volume, issue, pages, publisher, URL, access date, and others depending on type. Missing elements prevent location and verification of the source. Verify all required elements are present for each entry.

For each reference entry, identify the source type and the elements the style requires. Check that every required element is present. Common missing elements include page numbers for articles, access dates for websites, edition numbers for books, and DOI for scholarly articles. Flag incomplete entries and recommend the missing elements be added. Complete entries allow readers to find the source through multiple paths, which is essential when one path, like a URL, fails. Missing elements are a common completeness failure that undermines traceability.

### Check Element Accuracy And Consistency

Beyond presence, bibliographic elements must be accurate and consistent. Author names must be spelled correctly and consistently. Dates must match the source. Titles must be accurate. Journals and publishers must be correctly named. Check element accuracy and consistency across the bibliography.

For each entry, verify the elements against the source where possible. Check author name spelling and formatting for consistency across entries by the same author. Verify dates match the source, distinguishing publication, online, and access dates. Verify titles are accurate and not abbreviated inconsistently. Verify journal and publisher names are correct and consistently styled. Inaccurate or inconsistent elements make sources hard to find and signal inattention. Accuracy checking catches errors that completeness checking, which only verifies presence, misses.

### Verify Source Type Identification And Formatting

Each reference entry must be formatted according to its source type, which requires correctly identifying the type. A website formatted as a journal article, or a chapter formatted as a book, will be nonconforming and potentially misleading. Verify source type identification and formatting.

For each entry, verify the source type is correctly identified. Check that the format matches the identified type per the style guide. Be alert to ambiguous types, such as online reports, preprints, or institutional publications, which may be formatted differently depending on the style. Flag misidentified or misformatted entries. Correct source type identification ensures each entry follows the right format and provides the right elements for its type.

### Check For Missing Source Categories

Some works should include source categories beyond the main reference list: primary sources, archival sources, legal sources, datasets, or multimedia. If the work uses these source types, verify they are documented in the appropriate apparatus, whether the main list or a separate section. Check for missing source categories.

Survey the source types used in the work. Verify that each type is documented in the bibliography or an appropriate separate list. Check that primary or archival sources are listed if the style or discipline requires. Check that legal sources, datasets, or multimedia are documented per the relevant conventions. Flag source categories that are used but not documented. Complete documentation covers all source types the work relies on, not just the common ones.

### Verify Bibliographic Completeness Across Editions And Versions

For works that go through multiple editions or versions, the bibliography must remain complete across revisions. Sources added in revision must be added to the bibliography; sources removed must be removed. Drift between editions creates incomplete apparatus. Verify completeness across versions.

During revision, verify that bibliography changes keep pace with citation changes. After any revision that adds or removes citations, check that the bibliography is updated correspondingly. For new editions of existing works, verify the bibliography reflects the current edition's sources, not a stale version. Bibliographic drift is a common completeness failure in revised works, and it accumulates across editions. Active verification during revision prevents drift.

## Common Traps

### Orphan Citations Or References

Every citation needs an entry and vice versa. Cross-reference both directions.

### Missing Required Elements

Each source type requires specific elements. Verify presence for every entry.

### Inaccurate Or Inconsistent Elements

Names, dates, and titles must be accurate and consistent. Check against sources.

### Misidentified Source Types

Wrong type leads to wrong format. Verify identification before checking format.

### Missing Source Categories

Primary, legal, data sources may need separate documentation. Check coverage.

### Bibliographic Drift Across Revisions

Citation changes must update the bibliography. Verify during revision.

### Checking Format But Not Completeness

Correct format with missing sources or elements still fails completeness.

## Self-Check

Before treating bibliographic completeness as complete, verify:

- Every source cited in the text appears in the bibliography or reference list.
- Every source listed in the bibliography is cited in the text or explicitly designated.
- Each reference entry contains all required elements for its source type per the style guide.
- Bibliographic elements are accurate, matching the sources.
- Author names, dates, and titles are consistent across entries and mentions.
- Source types are correctly identified and formatted accordingly.
- All source categories used in the work are documented in the appropriate apparatus.
- The bibliography reflects the current version, with no drift from revision.
- Cross-referencing between citations and bibliography is complete in both directions.
- The scholarly apparatus fully and consistently documents the work's sources.
